### 2. Omar's framing: Terraform is below level one

Omar's core position: the next six months are not about an ambitious AI north star. They are about basic primitives that do not yet exist:

- Terraform CLI cannot effectively run agents.
- Coding engines can only partially access TFC/TFE data.
- Resource data, relationship data, version data, module registry data, private registry data, and workspace context are not reliably available to agents.
- MCP does not yet solve the real workflow gaps.
- Users cannot chat with Explorer or workspace context.

**Module registry gap (clearest example):** A large share of revenue and customer usage depends on modules. Private registry modules often contain 30-50 resources. Current MCP-style lookup is tied mainly to module names - insufficient. Agents need semantic summaries of module purpose, resource composition, and applicability so that a prompt like "S3 with RDS" can surface the right internal modules.

### 3. The product ladder

Omar articulated a three-level progression. The six-month focus is level one only.

**Level one: primitives / table stakes (6-month focus)**
- Agent-capable Terraform CLI
- TFC/TFE data availability to coding engines
- Usable registry and private module context with semantic summaries
- Workspace / resource / relationship / version context
- Explorer/chat context
- MCP gaps closed or compensated for

**Level two: upgrade and repair agents (downstream of level one)**
- Terraform version upgrade agent
- Provider version upgrade agent
- Module version upgrade agent
- Plan-error repair
- Sentinel policy repair
- Terraform policy / run-error repair
- Proactive workspace scanning against new provider/Terraform releases
- Safe auto-plan / suggested upgrade flows

**Level three: AI SRE / AI DevOps / incident remediation (2027-2028)**
- Hook into logs and observability systems (Datadog, etc.)
- Listen for events or incidents
- Trigger Terraform workflows for remediation
- Connect Slack/docs/bots and infrastructure change flows

**Decision:** Do not reject levels two and three. Keep them downstream of level one. The prerequisite layer is missing.
