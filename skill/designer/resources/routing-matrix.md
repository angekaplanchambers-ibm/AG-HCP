# Designer Routing Matrix

| Prompt shape | Route | Fidelity |
|---|---|---|
| What method should I use? | design-method-finder | process |
| I am stuck / what next? | design-method-finder | process |
| Remote paper prototype / participatory design | design-method-finder | process or low-fi prototype |
| Persona / JTBD / CUJ | product-designer | product framing |
| OOUX / object model / IA / flow / wireframe | product-designer | structure |
| Loading / validation / error / undo / focus / timing | microinteractions-expert | micro-spec |
| Make it beautiful / polished / production-grade UI | impeccable | visual/UI craft |
| Review / critique / evaluate existing artifact | critique if installed; product-designer fallback | critique |
| Dual critique / dual crit / dual-lens critique | critique if installed; product-designer two-pass fallback | critique |

When in doubt, ask: `What decision are you trying to make next?`
