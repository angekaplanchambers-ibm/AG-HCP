# design-method-finder

Design method recommendation skill for Pocket Product Designer v2.

Use it when the question is how to approach the work, not when the user already knows which artifact they need.

## Example prompts

```text
What method should I use to validate this onboarding flow?
```

```text
We need users to help design the dashboard, not just test it.
```

```text
I like paper prototypes, but this team is remote. What should I do?
```

## Output

- Situation summary
- One primary method
- Runnable session plan
- Materials
- Expected outputs
- Watchouts
- Fallback and upgrade path
